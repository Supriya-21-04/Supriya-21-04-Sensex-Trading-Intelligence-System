Build a web platform that helps traders analyze their strategies, find hidden patterns, and generate trading code — powered by AI.
Goal 1 — Build the Backtest Analyzer
What it does: Traders upload their trade history and instantly get deep analysis
Success looks like:

User uploads a CSV of their trades
Platform shows win rate, PnL curve, max drawdown, Sharpe ratio
Monte Carlo simulation tells them if their edge is real or luck
Identifies which market conditions their strategy works best in


Goal 2 — Build the AI Signal Tester
What it does: Traders describe a trading idea in plain English and AI tests if it actually works
Success looks like:

User types "does my strategy win more when volume is high?"
Platform tests it against their data instantly
Returns clear yes/no with statistical confidence score
Suggests related patterns worth exploring


Goal 3 — Build the Pine Script Generator
What it does: AI writes TradingView code from plain English descriptions
Success looks like:

User types "write a strategy that buys when RSI crosses 30 and ATR is rising"
Platform generates working Pine Script instantly
Auto-fixes any errors without user needing to debug
User drops it into TradingView and it works first time


Goal 4 — Build the ML Factor Analyzer
What it does: Machine learning finds hidden patterns in trade data that humans miss
Success looks like:

Automatically tests 50+ market variables against trade outcomes
Reveals which combination of factors predicts winning trades
Shows feature importance in simple visual charts
Gives actionable insights not just raw numbers


Goal 5 — Build the Prop Firm Simulator
What it does: Simulates whether a strategy would pass a funded account challenge
Success looks like:

User inputs their strategy stats and challenge rules
Platform runs thousands of simulations
Shows probability of passing, failing, hitting drawdown limits
Suggests how to adjust strategy to improve odds
